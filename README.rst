chaordic-code-challenge
=======================

Chaordic - Code Challenge

Requirements
------------

- `PHP <http://php.net/>`_ >= 5.5.12

Usage
-----

In order to run it locally, make use of PHP's built-in web server in the
project's root path:

.. code-block:: bash

   $ php -S localhost:8000

Deployment to Heroku
--------------------

.. code-block:: bash

   $ ./deploy.sh

-------
